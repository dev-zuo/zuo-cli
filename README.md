# zuo-template
Vue2、Vue3、Vitepress、React、Koa 等框架初始化项目、最小化测试 demo 收集。方便测试、学习框架基本功能/特性。

## 为什么会有这个项目
- 当项目中发现某些代码无法正常运行时，排查问题的最好方法是使用最小化测试 demo。收集初始化项目模版，可以减少类似 vue create xx 生成初始化项目时间，提高 debug 效率。
- 对于一些
